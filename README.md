# Presentation-timer
学会のプレゼンなどで使えるタイマー

# 特徴
- 1鈴、2鈴、3鈴を鳴らせます。1鈴後はタイマーが黄色、2鈴後はタイマーが橙色、3鈴後はタイマーが赤色に変わります。
- カウントアップ／カウントダウンを指定できます（カウントダウンの場合は、1～3鈴のいずれかのうち、最も長い時間を指定している時間が、カウントダウン開始時の残り時間になります）。
- GETパラメータで初期設定を指定できます。
  - `b1`：1鈴の時間。`mm:ss`で指定
  - `b2`：2鈴の時間。`mm:ss`で指定
  - `b3`：3鈴の時間。`mm:ss`で指定
  - `mode`：カウントアップかカウントダウンかを指定。`up`（デフォルト）でカウントアップ、`down`でカウントダウンです。
  - 例）`?b1=10:00&b2=12:00&b3=15:00&mode=down` で、1鈴10分、2鈴12分、3鈴15分のカウントダウンタイマーを設定します
- PWAに簡易的に対応しています。アプリとしてインストールすると（iOSでは）フルスクリーンで利用できます（Androidはアプリとしてインストールできませんが、Chromeで開いた場合、UIが消えるようにスクロールすることで画面いっぱいにタイマーを表示できます）。
